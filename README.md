### Prerequisites

* Node.js (>= 18.x)  
* Expo CLI (`npm install -g expo-cli`)  
* Git  

### Steps

1. Clone the repository:

   ```bash
   git clone https://github.com/lihiniapsara/DailyReflectionApp.git
   cd DailyReflectionApp
````

   ```bash
   npm install
   # or
   yarn install
   ```


   * Scan QR code in **Expo Go** app (iOS/Android)
